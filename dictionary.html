Chapter 20

Layout Formating

Box Model
Is the padding, boarder and margin around an object

Backgrounds and background images can be assigned to these items as well

Float
Moving an object around within the parent object.
Position
Change an objects location in the browser window without changing the location of your code.
Z-Index
Moves the object from back to front.

The mouse cursor can also be changed, but the book cautions against it as it might make the user believe their system is gridlocked.
